Was Emacs 23 bringen wird...

Emacs has a new option —daemon that runs emacs on the background as a server, so you only need to call emacsclient

Genial. Zumal schon jetzt meine .bashrc sagt:

   EDITOR=/usr/bin/emacsclient
   ALTERNATE_EDITOR=emacsnw

Wobei emacsnw eigentlich ein alias zu emacs -nw ist, der ein Emacs im Terminal startet. Der —deamon Modus wird in Emacs 23 dann sogar nicht unterscheiden, ob ein TTY-Emacs oder ein graphischer (plain oder GTK) am laufen sind. In beiden k√∂nnen dann dieselben Buffer bearbeitet werden.
