Screenshots aus der Konsole

Klicki-Bunti-Tools sind was für Looser… Unter Linux macht man einen Screenshot so:

   import -window root screenshot.png

Dazu brauch man eigentlich nur ImageMagick, import ist da mitbei.

In meinem Window-Manager kann man den Befehl einfach auf eine Taste bzw. Tastenkombination legen.

Demnächst dann mehr in diesem Kino…
