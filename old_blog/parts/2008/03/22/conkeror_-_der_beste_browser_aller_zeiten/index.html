Conkeror - Der beste Browser aller Zeiten

Conkeror is a highly programmable web browser for the Mozilla XULRunner platform. Inspired by Emacs, it has a sophisticated keyboard system for running commands and interacting with webpage DOM. It can interface with external editors and viewers. It is self-documenting and extensible with Javascript.
