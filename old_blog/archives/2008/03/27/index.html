Songbird 0.5 Release

Songbird ist zwar nicht so “my cup of tea” wenn es um einen Mediaplayer geht, aber ich kenne einige, die den Vogel jetzt schon lieben und es gibt auch wirklich ein paar äußerst innovative Features. Die Integration von Web und Musik ist in keiner anderen Software so gut gelungen, schließlich wollen die Macher ja den “Firefox für Musik” erschaffen.

Das neue Release hat wieder tonnenweise Bugfixes und einige neue Features. Details dazu gibts im Songbird Blog.

Download, Dokumentation und der ganze Rest: http://www.songbirdnest.com

Klicki-Bunti-Tools sind was für Looser… Unter Linux macht man einen Screenshot so:

Dazu brauch man eigentlich nur ImageMagick, import ist da mitbei.

In meinem Window-Manager kann man den Befehl einfach auf eine Taste bzw. Tastenkombination legen.
