Numark M101USB & Linux

Numark M101USB – Budget DJ Mixer mit USB Audio Interface (Aufnahme & Wiedergabe). Systemvorraussetzung laut Numark Website: Windows oder Mac. Dort aber auch zu lesen, dass es sich um ein Class Compliant Gerät handelt – d.h. unter Windows bzw. Mac sind keine speziellen Treiber nötig. Natürlich werden USB Audio Class Compliant Geräte aber auch unter Linux durch Alsa unterstüzt (siehe z.B. Linuxaudio.com oder ALSA Wiki).

Konfiguration unter openSUSE 12.3 ist erfreulich unkompliziert: Einstöpseln –> YaST –> Soundkarte –> USB Soundkarte (unkofiguriert) auswählen –> Konfigurieren –> Automatisch –> Fertig!

Dann kann man z.B. mit Audacity heiße Mixsessions in 1A Qualität aufnehmen (Stay Tuned!).

JabRef

Bibliographieverwaltung vom Feinsten für den (Natur-)Wissenschaftler. Plattformunabhängig dank Java & dank flexibler Plug-In Architektur aufrüstbar. Für den Astrophysiker besonders interessant ist das ADS Plugin, mit dem deren BibTeX Einträge einfach importiert werden können.
