flickrfavsaver.sh

Bildschirmschoner, wie z.B. xscreensaver, bieten einen Diashow-Modus, bei dem nach Aktivierung des Bildschirmschoners nacheinander (und wahlweise zufällig) Bilder aus einem bestimmten Verzeichnis angezeigt werden.

Mit meinem Bash-Script flickrfavsaver.sh ist es möglich, ein Verzeichnis automatisch mit Fotos aus einem Flickr!-Favoriten Stream zu befüllen. Dabei achtet das Skript zum sowohl darauf, dass Bilder, die nur in geringer Auflösung herunterladbar sind, nicht im gewählten Verzeichnis landen, als auch darauf, dass alte Bilder nach einer gewissen Zeitspanne wieder gelöscht werden und somit das Verzeichnis nicht vollgemüllt wird.

Lässt man flickrfavsaver.sh mit seinen persönlichen Flickr!-Favoriten regelmäßig als cronjob auf seinem Rechner laufen und wählt man nun für den Diashow-Modus seines Bildschirmschoners das entsprechende Verzeichnis aus, so hat man einen wahrlich individuellen Bildschirmschoner.

Link zum Script: http://knusper.net/flickrfavsaver/flickrfavsaver.html

(Hinweis: Benötigt xsltproc & die Datei parse_enclosure.xsl)

qpdfview

Einfacher PDF Betrachter mit Reitern (aka Tabs). Basiert auf Qt4, fügt sich also prima in eine KDE 4 Arbeitsumgebung ein. Falls man schnell(!!!) durch hunderte PDFs mit Diagrammen navigieren muss (siehe Screenshot) mehr als ideal.

Homepage: https://launchpad.net/qpdfview

Kann bald (ab Version 0.4.7) auch in PDF eingebettete Filme starten. Ein Feature, welches ich in der aktuellen, stabilen Version 0.4.6 noch vermisse.

Pakete für openSUSE (oder Fedora, oder CentOS) gibt es im openSUSE Build Service QT Desktop Repository.

Matplotlib Animation as Animated GIF

First save matplotlib animation in python code with:

ani.save('animation.mp4', fps=10)

Use ffmpeg to decode single frames to gif:

mkdir gifs && ffmpeg -i animation.mp4 -r 10 gifs/%04d.gif

Use gifsicle to create animated gif:

gifsicle --use-colormap 'bw' --delay=10 --loop ./*gif > animation.gif

See also this entry at dtbaker.net/random-linux-posts.
