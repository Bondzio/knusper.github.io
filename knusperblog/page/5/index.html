15 Essential Firefox Add-Ons - I use 'em every day!

Firemacs

Emacs key bindings in Firefox.

http://www.mew.org/~kazu/proj/firemacs/en/

Classic Theme Restorer

Revert to the old UI, that Firefox had before they launched the Australis interface with version 29.

https://addons.mozilla.org/de/firefox/addon/classicthemerestorer/

DownThemAll

Most advanced download manager for Firefox.

http://www.downthemall.net/

Firebug

Web development essentials – edit HTML and CSS and see the changes in real time!

https://www.getfirebug.com/

FireFTP

FTP Client for Firefox.

http://fireftp.net/

Fox Clocks

World time clock in your Firefox toolbar / statusbar.

https://foxclocks.org/

Greasemonkey

Miss that download button on Youtube? Want to use your own key bindings on web page xyz. Remix the web, by running customized scripts on websites.

http://www.greasespot.net/

It’s All Text!

Edit text fields in forms with your favorite text editor.

https://addons.mozilla.org/de/firefox/addon/its-all-text/

Menu Editor

Customize Firefox’s menus.

http://menueditor.mozdev.org/

OverbiteFF

Support for gopher:// protocol.

http://gopher.floodgap.com/overbite/

Pinboard Extension

Add support for Pinboard bookmark manager.

https://pinboard.in/resources/

Quick Proxy

Use a proxy from time to time? Turn it on and off with just a simple click!

http://ozzie.io/category/quickproxy/

Status-4-Evar

Status and progress bar for Firefox. Olschool!

http://forums.mozillazine.org/viewtopic.php?f=48&t=2001965

Stylish

Apply custom CSS to websites.

https://userstyles.org/

Tab Mix Plus

100+ tabs open? No problem…

http://tmp.garyr.net/

bash autocd

To save typing, set in your .bashrc

shopt -s autocd

Now, you just do directory instead of cd directory.

Works with bash versions > 4. For more cool stuff, check Section 4.3.2 of bash manual.
