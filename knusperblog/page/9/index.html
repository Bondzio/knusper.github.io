Discogs, Google & DuckDuckGo - Dark Themes

I prefer using a dark theme on my desktop. However, now most of the brighter styled web-sites look “alien” when opened in my browser. The solution is “re-theming” those sites using custom CSS styles. The Firefox (or Chrome) plug-in Stylish helps to manage such custom styles, and ten-thousands of styles are available on userstyles.org.

Cubehelix as Default in ds9

I love Dave Green’s cubehelix color scheme for visualization of astronomical intensity datasets (e.g. images).

In fact, I love it so much, that I want it to be the default in SAOImage ds9. A little alias does the trick for me:

alias ds9='~/local/bin/ds9 -cmap file ~/local/lib/cubehelix00.ds9'

Here the file cubehelix00.ds9 must be obtained from this page.
