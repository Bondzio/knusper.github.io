Magic Tokens (2010 ZX Spectrum 48K Game)

Magic Tokens, schon in meinem alten Blog gefeatured, ist ein wunderbares “Match-3” Game aus dem Jahr 2010 für den Sinclair ZX Spectrum (48KB RAM). Achtung: Löst den “Ach, eine Runde Spiel ich noch – und plötzlich ist es morgens halb 5”-Effekt aus.

Download: http://shiru.untergrund.net/files/zx/magic_tokens.zip

Magic Tokens @ World of Spectrum

Falls man keinen Speccy rumstehen hat, braucht man natürlich einen Emulator. Getestet und für gut befunden habe ich Fuse – the Free Unix Spectrum Emulator. Es gibt aber auch einen Java Emulator, mit dem man direkt im Browser loszocken kann.
