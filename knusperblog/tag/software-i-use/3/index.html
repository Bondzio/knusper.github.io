ProjectM

Soundvisualisierungsding (i.e. “Trippy Eye Candy”).

Im Prinzip die Opensource Version vom bekannten Winamp Plugin Milkdrop.

Link: http://projectm.sourceforge.net/

Lieblings-Feature: Mittels verstellbarer Pilzgröße (!) kann man den Beliebtheitsgrad der geladenen “Presets” markieren.

Pro-Tip: In openSUSE 12.3 gibt es ein funktionsfähiges offizielles Paket (nachdem ich einen Bug meldete, der das alte Paket für 99.9% der Benutzer unbenutzbar machte).

Dragonplayer

Minimalististischer Videoabspieler für KDE4. Die schlanke Alternative zu Kaffeine.

Wikipedia: de.wikipedia.org/wiki/Dragon_Player

Dragonplayer als “default” Videoabspieler unter KDE4 geht folgendermaßen:

Systemeinstellung –> Dateizuordnungen –> Video und dann bei allen gewünschten Videoformaten Dragonplayer nach ganz oben in der Rangfolge schieben.
