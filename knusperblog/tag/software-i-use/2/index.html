Firefox - GTK Dark Theme Fix

Problem:

Using a dark KDE (or GTK native) color theme (e.g. zenburn) causes content in form input elements to be illegible. This known Firefox bug basically renders the browser unusable on many webpages, that do not come with custom CSS rules for their forms.

Solution:

Use the Stylish Firefox add-on to create a custom global rule no dark forms and buttons:

    input:not(.urlbar-input):not(.textbox-input) {
        -moz-appearance: none !important;
        background-color: white;
        color: black;
    }

    #downloads-indicator-counter {
        color: white;
    }

    textarea {
        -moz-appearance: none !important;
        background-color: white;
        color: black;
    }

    select {
        -moz-appearance: none !important;
        background-color: white;
        color: black;
    }

Succsesfully tested with Firefox version 31.

Source: Arch Linux Wiki – Retrived 19.08.2014

Cubehelix as Default in ds9

I love Dave Green’s cubehelix color scheme for visualization of astronomical intensity datasets (e.g. images).

In fact, I love it so much, that I want it to be the default in SAOImage ds9. A little alias does the trick for me:

alias ds9='~/local/bin/ds9 -cmap file ~/local/lib/cubehelix00.ds9'

Here the file cubehelix00.ds9 must be obtained from this page.

KRename

Aufgabenstellung: Eine größere Anzahl von Dateien soll mittels eines bestimmten Musters umbenannt werden.

Die Hackerlösung besteht jetzt natürlich darin, einen Shellbefehl mit regulären Ausdrücken eintippen, vielleicht sogar unter Benutzung eines kleinen Loops und der eingebauten Shellarithmetik. Für alle anderen bietet sich KRename an. Tatsächlich ist das Programm sogar in der Lage bestimmte Informationen, welche für das Umbenennungsmuster gebraucht werden könnten, direkt aus einer Vielzahl von verschiedenen Dateiformaten auszulesen (z.B. MP3, PDF, JPG usw.). Das wäre auf der Shell auch nur unter Benutzung von zusätzlichen, teils exotischen, Tools möglich.

http://www.krename.net/

Wikipedia: https://en.wikipedia.org/wiki/KRename

Einfacher PDF Betrachter mit Reitern (aka Tabs). Basiert auf Qt4, fügt sich also prima in eine KDE 4 Arbeitsumgebung ein. Falls man schnell(!!!) durch hunderte PDFs mit Diagrammen navigieren muss (siehe Screenshot) mehr als ideal.

Pakete für openSUSE (oder Fedora, oder CentOS) gibt es im openSUSE Build Service QT Desktop Repository.

Matplotlib Animation as Animated GIF

First save matplotlib animation in python code with:

ani.save('animation.mp4', fps=10)

Use ffmpeg to decode single frames to gif:

mkdir gifs && ffmpeg -i animation.mp4 -r 10 gifs/%04d.gif

Use gifsicle to create animated gif:

gifsicle --use-colormap 'bw' --delay=10 --loop ./*gif > animation.gif

See also this entry at dtbaker.net/random-linux-posts.
